# rails_jwt_admin
> Jwt for rails admin.

## installation
```shell
# gem
gem install 'rails_jwt_admin'

# rails 
# 1. add gen & bundle install
gem 'rails_jwt_admin'

# 2. install admin
rails g rails_jwt_admin:install

# 3. check talbe & rake db:migrate
rake db:migrate
```
